# Offline Multimodal RAG Assistant

## Project Overview
Offline Multimodal RAG Assistant is a PDF-based question-answering system built using Python and Streamlit.

The project allows users to:

- Upload PDF files
- Ask questions related to uploaded PDFs
- Extract text from documents
- Display answers in an interactive interface

---

## Features

### PDF Upload
Upload PDF documents directly through the interface.

### Question Answering
Ask questions related to uploaded content.

### Text Extraction
Reads and processes PDF text.

### Cloud Deployment
Deployed using Streamlit Cloud.

### Interactive UI
Simple and clean interface.
